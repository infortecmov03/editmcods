# Roadmap

Este documento descreve os objetivos e os recursos planejados para as futuras versões do projeto.

## Versão Futura

### Editor Principal
- [ ] Implementar a funcionalidade da tecla "Tab" para inserir espaços.
- [ ] Adicionar um seletor de linguagem para o destaque de sintaxe.
- [ ] Sincronizar a rolagem horizontal entre o `textarea` e o `Highlighter`.
- [ ] Melhorar o feedback visual para o usuário (números de linha, etc.).

### Qualidade de Código
- [x] Configurar ESLint para análise estática de código.
- [x] Configurar Prettier para formatação automática.
- [ ] Adicionar Husky para executar hooks de pré-commit (lint e formatação).

### Funcionalidades Adicionais
- [ ] Salvar o estado do editor no `localStorage`.
- [ ] Adicionar um botão para copiar o código para a área de transferência.
- [ ] Implementar temas (claro e escuro).
