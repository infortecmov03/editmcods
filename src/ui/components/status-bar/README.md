# Diretório `status-bar`

Este diretório contém os componentes que compõem a barra de estado da aplicação, localizada na parte inferior da interface. A barra de estado é uma fonte de informação contextual importante para o utilizador.

## Ficheiros Principais

### `status-bar.tsx`
- **Descrição**: O componente principal que renderiza a barra de estado. Ele atua como um contentor para as várias informações e indicadores que são exibidos, como a posição do cursor, a linguagem do ficheiro, o estado do Git, etc.

### `index.tsx`
- **Descrição**: O ponto de entrada que exporta o componente `StatusBar` para ser integrado no layout principal da aplicação.
