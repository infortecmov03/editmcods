# Editor de Código Baseado em React

Este é um projeto de editor de código-fonte construído com React e TypeScript. A arquitetura é modular, separando o núcleo do editor da sua interface de usuário.

## Navegação do Código-Fonte

A estrutura principal do projeto está no diretório `src/`. Para entender a arquitetura em detalhes, comece pelos seguintes `README.md`:

- **[`src/README.md`](./src/README.md)**: Visão geral da arquitetura, explicando a separação entre o núcleo (`core`) e a interface do usuário (`ui`).

- **[`src/core/README.md`](./src/core/README.md)**: Detalhes sobre o núcleo do editor, que lida com a lógica de manipulação de texto, análise de código e realce de sintaxe.

- **[`src/ui/README.md`](./src/ui/README.md)**: Detalhes sobre a camada de interface do usuário, construída com componentes React.
